Este proyecto es un sistema para una agroquímica.

Reglas:
- presupuesto requiere teléfono obligatorio
- mail opcional
- ventas deben ser CON_FACTURA o SIN_FACTURA
- registrar usuario en cada operación
- caja con apertura y cierre
- cuenta corriente con saldo y vencimientos

Tecnología:
- Node.js
- Express
- Prisma
- PostgreSQL

Prioridad:
1. auth
2. usuarios
3. productos
4. clientes
5. presupuestos
6. ventas
7. caja
8. cuenta corriente

Mantener simplicidad.
