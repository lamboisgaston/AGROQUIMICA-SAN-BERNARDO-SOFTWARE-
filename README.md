# Agroquimica San Bernardo

Sistema operativo simple para agroquímica.

## Roles
- DUENO
- GERENTE
- CAJA
- MOSTRADOR

## Reglas
- No permitir presupuesto sin teléfono
- Mail opcional en presupuesto
- Toda venta debe ser CON_FACTURA o SIN_FACTURA
- Registrar siempre qué usuario hizo cada operación
- Caja con apertura y cierre
- Cuenta corriente con saldo, pagos y vencimientos
